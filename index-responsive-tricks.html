Media Queries

Syntax:

@media all and (min-width: 576px) { ... }

Commonly used properties:

min-width, max-width, min-height, max-height, orientation, resolution

Examples:

.mediaquery-fontsize { font-size: 10px; } .mediaquery-display { display: none; } .mediaquery-menu { min-width: 97%; }
@media all and (min-width: 576px) { .mediaquery-fontsize { font-size: 20px; } .mediaquery-display { display: block; } .mediaquery-menu { min-width: 20%; } }
